Getting a Certificate

You can use either a DOEGrid Certificate or a Fermi Kerberos (KCA) certificate.
Each certificate type will need to be separately registered with the database. To minimize confusion, we strongly recommend that you choose only one certificate type for access. We recommend that Fermilab and other Kerberos users choose the KCA certificate. This is more secure and will, at least at Fermilab, be required for access to other web sites.

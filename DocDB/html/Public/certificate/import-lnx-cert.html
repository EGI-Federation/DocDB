Import or Export a Certificate - Linux

To export or save a certificate:

  • For Linux Firefox:
    • open Edit -> Preferences -> Advanced -> Certificates -> Manage Certificates
    • highlight the certificate to save
    • click "Backup"
    • select the desired directory and file name
    • make sure the file type is PKCS12 (*.p12)
    • click "Save"
    • You will get a dialog box asking for a certificate backup password. Choose a simple password.
    • click "OK"
    • click "OK" on any remaining open dialog boxes
  • For Mozilla:
    • open Edit -> Preferences -> Privacy & Security -> Certificates -> Manage Certificates
    • highlight the certificate to save
    • click "Backup"
    • select the desired directory and file name
    • make sure the file type is PKCS12 (*.p12)
    • click "Save"
    • If you set a master password, you will get a dialog box asking for the password.
    • You will get a dialog box asking for a certificate backup password. Choose a simple password.
    • click "OK"
    • click "OK" on any remaining open dialog boxes

To import a saved certificate:

  • For Linux Firefox:
    • open Edit -> Preferences -> Advanced -> Certificates -> Manage Certificates
    • click "Import"
    • browse to select the file you saved
    • make sure you are importing a PKCS12 file
    • click "Open"
    • When you get the Password Entry Dialog box, enter the password you used when you saved the certificate.
    • click "OK"
    • click "OK" on any remaining open dialog boxes
  • For Mozilla:
    • open Edit -> Preferences -> Privacy & Security -> Certificates -> Manage Certificates
    • click "Import"
    • browse to select the file you saved
    • make sure you are importing a PKCS12 file
    • click "Open"
    • You might be prompted for a Master Password - use something you can easily remember.
    • When you get the Password Entry Dialog box, enter the password you used when you saved the certificate.
    • click "OK"
    • click "OK" on any remaining open dialog boxes

  • You can test your certificate if you have problems.

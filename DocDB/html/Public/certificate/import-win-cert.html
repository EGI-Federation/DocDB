Import or Export a Certificate - Windows

These instructions assume that you have a certificate saved as a file somewhere. You will need to know the location of that file and any password you used when creating the certificate. This is not your Kerberos or other user password.

  • For Mozilla:
    open edit -> preferences -> Privacy & Security -> Certificates -> manage certificates
    click import and enter the filename (/tmp/x509up_u<your uid>.p12)
  • For Windows Firefox:
    open tools -> options -> Advanced -> Certificates -> manage certificates
    click import and enter the filename
  • For Windows IE:
    open tools -> Internet Options -> Content -> Certificates
    click import and enter the filename
  • If you have problems, test your certificate
